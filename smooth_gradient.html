Smooth rainbow gradient

Every image is its own <section> so css linear-gradient() couldn't be used. Instead javascript was used to replicate linear-gradient() for an unknown amount of section elements. The javascript function interpolates between two colours for a section based on its top position in the document.
I also added the hearts animating from bottom to top when scrolling with a random css transform for different scale, rotation and x position.
The project could previously be found here: postersofpride.com. It was done for 2017 pride parade.
