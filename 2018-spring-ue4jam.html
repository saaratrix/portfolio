2018 Spring UE4 Gamejam

A game made for the 2018 Spring #ue4jam.
Theme: Transformation.
The game was made using Unreal Engine 4 and I did the game mechanics such as shooting a projectile that either restores or destroys objects. I also did the particle effects for the projectiles.
