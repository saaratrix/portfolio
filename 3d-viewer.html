A 3D model viewer

This is a 3D model viewer I made using three.js and angular to view 3d models models. The viewer can be seen in action here: https://saaratrix.github.io/generic-3Dproduct-viewer/build/

The first 3 models are rendered using Phong shading and the 4 furnitures are rendered using PBR metallic/roughness workflow.
