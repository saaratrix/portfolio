Website blur effects

Client work for the website chef-rory.com. The client wanted a blurring effect to transition between several things on the website. Many of the blur transitions were done using CSS transition. For the main page transition effect a custom animation had to be created to offer better control over CSS transition.

The main page transition. When you're on index.html and you mouseover Dessert Bar NYC it will over time blur the white background. And eventually swap to black background and unblur the dessert.html image. And vice-versa if you're on dessert.html

The image slideshow sets blur values at 0%, and 50% progress to first blur the visible image and then unblur the next image.
